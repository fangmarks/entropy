# Requirements

- Deno
- MongoDB
- Docker (Optional)

# How to run

1. Install Deno (and/or Docker Compose)
2. `sudo docker-compose up -d` to bring up Mongo and Mongo-Express
3. `deno run -A mod.ts` to start Entropy
